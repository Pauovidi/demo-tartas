# Casa Bruna Portfolio Demo Notes

## Nueva marca elegida
- Nombre: Casa Bruna
- Tagline: Horneado lento para sobremesas memorables.
- Dirección visual: mediterráneo elegante con acento cálido editorial.

## Qué se ha rebrandado
- Identidad de marca completa: naming, tagline, sistema verbal y metadatos.
- Sistema visual: paleta, tipografía, hero, tarjetas, header, footer, checkout y chat launcher.
- Catálogo: sabores demo, precios demo y assets SVG propios.
- Conversación: saludo, handoff, copies de ayuda, reserva y contacto.
- Legales y documentación visible del repo.

## Qué se mantiene de la base técnica
- Rutas principales de home, catálogo, ficha, checkout, FAQ, prensa y admin.
- Lógica de carrito, checkout, agrupación por sabores y relación entre formatos.
- Flujo del chatbot, memoria, handoff, recordatorios y endpoints.
- Base de Supabase y estructura general del proyecto Next.js.

## Datos reales sustituidos
- Marca original y narrativa comercial.
- Teléfonos, WhatsApp, email, Instagram, datos legales y referencias personales.
- Fotografías, logo, favicon y vídeo hero originales.
- Sabores y copys que podían identificar al negocio real.

## Limitaciones demo
- Los datos de contacto son ficticios y no representan una atención real.
- El checkout demuestra el flujo, pero esta versión está presentada como demo safe.
- El panel admin conserva la lógica operativa, aunque con sabores y mocks reescritos.
